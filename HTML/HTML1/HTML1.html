A linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de marcaçaõ de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Alguns dos recursos da linguagem HTML vistos até este momento:

Recursos Tags associados Descrição
Links < a > Criar links entre paginas
Listas < ul > e < li > Criar listas de itens ordenados
< ol > e < li > Criar listas de itens nao ordenados
Imagens < img > Inserir imagens
